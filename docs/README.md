# Voice Ledger — 프로젝트 문서 인덱스

음성 녹음 파일(.m4a/.mp4)을 타임스탬프 포함 텍스트(.txt)로 변환하는 macOS 개인용 앱.

| [ARCHITECTURE.md](./ARCHITECTURE.md) | 시스템 아키텍처 — 레이어 설계, 데이터 흐름, 파일 구조 |

## 핵심 결정 요약

| 항목 | 결정 |
|------|------|
| 플랫폼 | macOS 데스크톱 앱 (PyQt6) |
| STT 엔진 | OpenAI Whisper medium (로컬, 오프라인) |
| 대상 하드웨어 | Apple M4 Mac (MPS 가속) |
| 입력 형식 | m4a, mp4 |
| 출력 형식 | .txt — `[HH:MM:SS] 텍스트` |
| 지원 언어 | 한국어 + 영어 자동 감지 |
| 개발 언어 | Python 3.11+ |
